# Accessibly Wordpress Widget
Wordpress plugin built with React.

----

### Quick Start
```sh
# Install node module packages
npm i

# Install PHP-composer dependencies [It's empty]
composer install

# Start development mode
npm start

# To run in production
npm run build
```
